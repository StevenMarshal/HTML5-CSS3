## 3.2 select标签

列表框标签，此标签和option标签配合使用

示例代码：

	<!DOCTYPE html>
	<html lang="en">
	<head>
		<meta charset="UTF-8">
		<title>表单标签t</title>
	</head>
	<body>
		<form>
			<!-- 
				select标签称之为：下拉框标签，和option标签配合使用
			 -->
			<select name="province">
				<option value="1">湖北省</option>
				<option value="2">湖南省</option>
				<option value="3">江苏省</option>
			</select>
		</form>
	</body>
	</html>
